# 🚀 TaskFlow Pro

A modern and responsive productivity dashboard built using **HTML, CSS, and JavaScript**. TaskFlow Pro helps users efficiently manage daily tasks with an elegant glassmorphism interface, smart task tracking, and real-time progress monitoring.

---

## 🌟 Features

### ✅ Task Management
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Task completion toggle
- Task persistence using Local Storage

### 📂 Task Categorization
- Internship
- Study
- Work
- Personal

### 🎯 Priority Levels
- High Priority
- Medium Priority
- Low Priority

### 📅 Due Date Tracking
- Assign due dates to tasks
- Smart due-date status indicators
- Organized task management

### 🔍 Search & Filters
- Instant task search
- Filter tasks by:
  - All Tasks
  - Active Tasks
  - Completed Tasks

### 📊 Productivity Dashboard
- Total Tasks Counter
- Completed Tasks Counter
- Pending Tasks Counter
- Circular Progress Indicator
- Dynamic Progress Bar

### 🌗 Theme Support
- Dark Mode
- Light Mode
- One-click Theme Toggle

### 🔔 User Experience
- Toast Notifications
- Delete Confirmation Dialog
- Empty State UI
- Smooth Animations
- Responsive Design

### 🎨 Modern UI Design
- Glassmorphism Effects
- Animated Gradient Background
- Floating Gradient Blobs
- Interactive Dashboard Cards
- Mobile-Friendly Layout
